# cube-simulator
Realistic cube simulator for Ludo game. The code is in progress.
This code is in Czech language.

Open IDLE, choose "File", choose "Open" and choose this project called "Ludo cube simulator.py".
Then you can edit this project by your self.
When you are finish with editing choose "Run" and "Run module" or press F5 on your keyboard.
In the GUI window appears number. By pressing the button called "Hoď kostkou" the number will be changing like the realistic Ludo cube.
At the end press the button called "Konec hry". The program wil ask you who won. You have to type the winner's name to the column and then press "Ok" button. If you didn't want to end the game press "Cancel" button. If you press the "Ok" button the program will show you the gratulation message with the name of the winner.
